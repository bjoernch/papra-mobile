# Papra Mobile (Unofficial)

<p align="center">
  <img src="app/src/main/res/drawable/ic_launcher_foreground.xml" alt="Papra Mobile logo" width="120" />
</p>

An unofficial Android client for Papra. This app is **not affiliated with Papra**.

## Features

- API key login
- Organization browsing
- Document list, upload, download, and preview
- Tags and tagging
- Offline availability
- Bulk actions
- Activity timeline

## Install

You can install the latest APK from the GitHub Releases page.

- Releases: https://github.com/bjoernch/papra-mobile/releases

## Development

- Android Studio + JDK 17
- Build release APK:

```bash
./gradlew assembleRelease
```
