# Papra Mobile

Papra Mobile is an unofficial Android client for Papra that lets you browse, upload, tag, and manage your documents on the go. It uses your Papra API key to connect to your own instance.

**Unofficial app. Not affiliated with Papra.**

## Features

- Browse organizations and documents
- Upload files and scan documents to PDF
- Tag management, bulk actions, and search
- Offline downloads and offline library view
- Trash view and document activity

### Obtainium setup (automatic updates)

1. Install Obtainium from https://obtainium.imranr.dev/ and open it.
2. Tap **Add App**.
3. Choose **GitHub Release**.
4. Paste this repo URL: `https://github.com/bjoernch/papra-mobile`.
5. Set APK filter to `papra-mobile.apk` (or use regex `papra-mobile\\.apk$`).
6. Save. Obtainium will track new releases automatically.

## Setup

1. Open the app and enter your Papra base URL (e.g. `https://docs.example.com`).
2. Create an API key in your Papra account with the needed permissions.
3. Paste the API key and continue.

## Permissions

The app uses your API key. No credentials are stored on a server. All requests go directly to your Papra instance.
